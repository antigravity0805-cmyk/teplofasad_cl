# TEPLOFASAD - Сайт теплофасадной компании

## 📁 Структура файлов

```
teplofasad/
├── index.html      (основная страница)
├── styles.css      (стили и дизайн)
├── script.js       (функционал калькулятора)
└── README.md       (этот файл)
```

## 🚀 Быстрый старт

### 1. Открытие сайта локально
- Скачайте все файлы в одну папку
- Откройте файл **index.html** в браузере (просто двойной клик)
- Готово! Сайт полностью работает

### 2. Загрузка на GitHub
```bash
# В папке с файлами выполните:
git init
git add .
git commit -m "TEPLOFASAD website"
git branch -M main
git remote add origin https://github.com/ВАШ_USERNAME/teplofasad.git
git push -u origin main
```

### 3. Развертывание на Vercel
1. Перейти на vercel.com
2. Нажать "Add New" → "Project"
3. Выбрать репозиторий teplofasad
4. Нажать "Deploy"

**Сайт будет доступен по ссылке:** `https://teplofasad.vercel.app`

### Цены материалов в калькуляторе
В `index.html` найдите:
```html
<select id="material">
    <option value="3500">Минеральная вата - 3500 тг/м²</option>
    <option value="3800">Базальтовая вата - 3800 тг/м²</option>
    <!-- и т.д. -->
</select>
```

### Имена бригадиров
Найдите в разделе "team" в `index.html` карточки с именами и отредактируйте.

### Цвета дизайна
В `styles.css` в начале файла находятся переменные:
```css
:root {
    --primary: #d4773f;        /* Основной оранжевый */
    --secondary: #8b6f47;      /* Коричневый */
    --dark: #2c2416;           /* Тёмный */
    /* и т.д. */
}
```
